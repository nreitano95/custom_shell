Please extract all files from the zip folder to a directory on your local machine.

Once you are in the root directory, you can compile smallsh.c with the following bash: 
	gcc -std=c11 -Wall -Werror -g3 -O0  smallsh.c -o smallsh

To enter into the smallsh shell, execute the command line interface syntax is as follows: 
	smallsh
